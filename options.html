Twitch Chat Translator — Options

When enabled, the extension will translate each new chat message and inject a translated line under it.
DeepL requires an API key.
How to get a free DeepL API key
1) Create a DeepL account and pick DeepL API Free.
- Sign up: deepl.com/signup
2) Open your DeepL account page → API keys & limits.
- This is where you create/copy your Auth Key.
3) Paste the key into DeepL API key above.
4) Keep DeepL endpoint set to Free API unless you have a Pro plan.

Notes:
- DeepL API Free includes a monthly character limit (check “API keys & limits”).
- If the key ever leaks, create a new key in DeepL and replace it here.
